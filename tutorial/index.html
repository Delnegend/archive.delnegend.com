How to playback movies on this site

Download and install your favorite media player

These are some recommedations

  • VLC Media Player - multiplatform
  • KMPlayer - multiplatform
  • iina - macOS
  • PotPlayer - Windows
  • Media Player Classic (with K-Lite Codec pack) - Windows
  • MX Player - Android
  • …

Copy the link and paste to the media player

There are ways for each application for you to paste the URL down, for example

  • VLC for Android: More –> New stream
  • VLC for Windows: Media –> Open Network Stream… OR Ctrl + N
  • MX Player: menu icon (top-left) –> Network Stream
  • PotPlayer: Menu (top-left) –> Open –> Open URL… OR Ctrl + U
  • …

Notes:

  • Subtitles are included in GitLab, RawGit links.
  • Because of the file size limitation of GitHub, I can’t store a whole mkv so subtitles are provided in separated files.
  • jsDelivr links are basically GitHub links but faster.
  • For whatever reason, with BitBucket links, you can only paste them directly into browsers to download the whole file.
